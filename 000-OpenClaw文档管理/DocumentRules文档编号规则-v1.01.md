# 001 OpenClaw架设部署｜DocumentRules 文档编号规则｜v1.01

本文件定义 OpenClaw_vv 仓库内项目文档、版本化目录和角色包的命名方式。

## 1. 项目分类目录

每个项目或文档分类使用独立编号：

```text
{编号}-{项目名分类}
```

当前示例：

```text
000-OpenClaw文档管理
001-OpenClaw规划设计
002-OpenClaw部署进度
003-OpenClaw事故经验
```

编号表示项目分类，不表示阅读顺序。阅读顺序由仓库根目录 `README.md` 维护。

## 2. 文档文件命名

目录内版本化文档使用：

```text
{English文档名中文文档名}-v{文档版本号}.md
```

示例：

```text
FinalDesign最终设计-v1.02.md
DeploymentPlan部署方案-v0.02.md
```
