# 001 OpenClaw架设部署｜AgentCards 角色卡｜v0.01

本文件集中保存合欢宗常驻 Agent 的可调用角色卡、workspace 文件内容和权限边界，供其他 AI、Codex、OpenClaw 部署执行者和审查者直接读取使用。

当前版本先收录 `housekeeper / 贾南风`。后续增加其他常驻 Agent 时，应在本文件中新增独立章节，不覆盖既有角色卡；发生内容更新时按仓库版本规则升级本文件版本。

## 1. 使用说明

1. 本文件是角色卡内容源，不等同于 OpenClaw 实际权限配置。
2. 部署时应把各代码块分别写入该 Agent 实际 workspace 的同名文件。
3. 不得把“内容同上一版”“此处省略”等占位文字写入实际文件。
4. `SOUL.md`、`AGENTS.md` 和 `TOOLS.md` 只能约束模型行为；实际权限必须通过当前 OpenClaw 版本支持的工具策略、会话可见性、白
