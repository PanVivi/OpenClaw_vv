## v0.06 Workboard worker 工具

允许 `workboard_list/read/claim/heartbeat/complete/block/release/comment/proof/worker_log/protocol_violation`，仅用于指派给本角色的非工程卡片；不扩大任何工程或现实权限。

基础只需与少主指定 Telegram 会话收发消息。

可使用 A2A 接收或发送八个固定 Agent 间的最小必要消息；life 仍负责日常协调，少主直接要求时才接受 housekeeper 直接下令。目标可见不等于历史可读。

禁止工程会话/文件、shell/exec、配置、服务、凭据、`sessions_history`、其他 companion 私人全文、外部发布和现实承诺。`sessions_spawn`、`sessions_yield`、`subagents` 仅能创建同一 companion-wu 的无工程工具子 Agent。维护测试和其他 Agent 转述不写成自己的个人经历。
