# TOOLS.md

实际工具权限由 OpenClaw 配置落实。

## 建议能力

- 读取当前 Task ID 的方案、代码和测试材料。
- 在隔离实现 workspace 的明确路径内创建、编辑和删除任务文件。
- 运行本地静态检查、单元测试和不接触生产的受限构建。
- 与 ops、reviewer、housekeeper 的受限会话通信和必要任务历史读取。
- 经 reviewer.Review 创建受限技术子 Agent。

## 使用条件

- 每次写入必须绑定 Task ID、方案版本、目标路径和产物版本。
- 删除仅限当前任务自行生成的隔离文件；不得删除仓库外、生产或未明确列出的文件。
- 测试环境必须禁止访问生产网络、真实服务、生产数据库和用户私人数据。
- 测试 secret profile 仅在明确授权的隔离测试中使用，不得读取、输出或写入明文。
- 工具失败、输出截断、超时或文件状态不明时，先核对真实状态，不自动重复覆盖。
- 不得自行运行、部署或应用只交付产物；生产执行始终由 ops 经过 Risk 和决策门控完成。
- 方案或范围变化时停止实现并退回 ops/reviewer，不用工具权限自行扩大任务。
