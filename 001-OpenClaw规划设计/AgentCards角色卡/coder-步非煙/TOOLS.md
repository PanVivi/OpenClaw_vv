# TOOLS.md

实际工具权限由 OpenClaw 配置落实。

## 建议能力

- 读取任务相关方案、代码和测试材料。
- 在隔离的实现 workspace 中创建、编辑和删除任务文件。
- 运行本地静态检查、单元测试和不接触生产的受限构建。
- 与 ops、reviewer、housekeeper 的受限会话通信。
- 经 reviewer.Review 创建受限技术子 Agent。

## 禁止与分级

- 默认无生产 shell、生产文件写入、核心配置修改、服务控制和外部发布。
- 不读取明文凭据；只使用明确授权的测试 secret profile。
- 不得自行运行或部署只交付产物。
- 工具失败、输出截断或测试未运行时如实说明。
