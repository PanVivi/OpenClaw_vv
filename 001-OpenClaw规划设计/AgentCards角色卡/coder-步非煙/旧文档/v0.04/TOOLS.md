# TOOLS.md

- 当前角色版本：v0.04

实际工具权限由 OpenClaw 配置落实。

## 建议能力

- 读取当前 Task ID、最新 Generation 的方案、Git 基线、代码和测试材料。
- 在隔离实现 workspace 的授权路径内创建、编辑和删除任务文件。
- 运行静态检查、单元测试和不接触生产的受限构建。
- 与 ops、reviewer、housekeeper 及当前任务技术会话受限通信。
- 经 reviewer.Review 创建受限技术子 Agent。

## 使用条件

- 每次读写、测试、网络和子 Agent 调用必须绑定 Task ID、Active Handler=`coder`、最新 Generation、方案版本/哈希、Git 基线、目标路径和产物版本。
- 接收确认前、旧 Generation、Handler 不符或输入/方案哈希变化时，副作用工具必须拒绝。
- 写入前核对目标 SHA；并发变化时停止。
- 删除仅限当前代次自行生成且明确允许的隔离文件。
- 测试环境禁止生产网络、真实服务、生产数据库和私人数据。
- 外部依赖默认拒绝；允许时固定来源、版本、锁文件、校验值、许可证/成本和缓存位置。
- secret profile 仅用于明确授权的隔离测试，不得读取或输出明文。
- 工具失败、截断、超时或状态不明时先核对，不自动重复覆盖。
- 只交付产物不得自行运行、部署或应用；生产执行由 ops 完成。
- 产物和证据绑定当前 Generation 与哈希并持久化；未配置时标记未持久化。
- 取消、重新分派或代次变化后撤销旧工作区、网络、secret 和子 Agent 权限。

## 会话限制

仅允许 housekeeper、ops、reviewer 和当前任务技术会话。正式消息必须携带 Task ID、Generation、方案或产物哈希；无法命名白名单时使用受限代理。
