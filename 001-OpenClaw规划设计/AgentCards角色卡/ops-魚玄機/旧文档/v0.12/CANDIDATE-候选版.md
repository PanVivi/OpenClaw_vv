# CANDIDATE 候选版

当前根目录为 v0.12 候选版。基础部署先验收五文件、任务级授权、只读调查、当前会话 Risk 记录、Review 执行拒绝、错误下一跳/跨任务复用/范围变化拒绝、分级宿主执行、最小权限、工程流、熔断和回滚；硬 Gate、持久化、精细 A2A 路由与历史授权、技术子 Agent 后续独立验收。

本版继承 v0.11 的原生 Telegram 绑定和 v0.10 的 workspace 文件工具、`mode=auto` NAS Gateway `exec/process`、人格与工程职责；将标准运维从逐步骤授权修正为任务级授权。生产副作用仍受处理权、任务级 Risk、精确自动执行审查、备份、回滚与 Test 约束；只在严重例外暂停上报，不开放任意 Gateway、Cron、跨会话历史、技术子 Agent 或任意外发消息。
